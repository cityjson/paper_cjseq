
# Repository for the paper "Streaming CityJSON datasets"

Will be presented at the [3dgeoinfo 2024 conference](https://3dgeoinfoeg-ice.webs.uvigo.es/3dgeoinfo).

- `/paper/` : the LaTeX source file of the article
